# AuthApp

Simple authentication system built with .NET Core, PostgreSQL, and Entity Framework Core. 

## Features
- User Registration
- User Login
- Data storage with PostgreSQL
- Secure password hashing

## Technologies Used
- C# (.NET Core)
- PostgreSQL
- Entity Framework Core
- ASP.NET Core
